# まのさば＋

[English](README.md) | [简体中文](README_CN.md)

-----

This project aims to enhance the limited visual presentation of "Magical Girl's Witch Trial"

-----

### Build & Run

```powershell

# Generate build files
cmake -B Build

# Build project (Debug configuration by default)
cmake --build Build

# Or build Release configuration (optimized, smaller size)
cmake --build Build --config Release

# Run Debug version
.\Build\Bin\Debug\manosaba_plus.exe

# Run Release version
.\Build\Bin\Release\manosaba_plus.exe

```
